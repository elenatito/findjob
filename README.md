# 🤖 AI Job Assistant

Локальный AI-помощник для поиска IT-вакансий с анализом релевантности, адаптацией резюме и ежедневной рассылкой.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📊 **Анализ вакансии** | Оценка соответствия от 1 до 10, список плюсов и минусов |
| 📝 **Адаптация резюме** | Переписывает резюме под конкретную вакансию |
| 🎯 **Подготовка к собеседованию** | Генерирует технические и поведенческие вопросы |
| 🔍 **Поиск вакансий** | Поиск на hh.ru с фильтром по опыту |
| 🏢 **Компании** | Мониторинг вакансий в интересных компаниях |
| 💰 **Анализ зарплаты** | Рыночная вилка, советы по переговорам |
| ✉️ **Cover Letter** | Автоматическая генерация сопроводительных писем |
| 📋 **Трекер откликов** | Статусы, напоминания, статистика |
| 📈 **Тренды рынка** | Востребованность технологий, зарплаты по грейдам |
| 📧 **Ежедневная рассылка** | Утром на почту — только релевантные вакансии (оценка ≥7) |
